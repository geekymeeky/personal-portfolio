# Overview
At Prepfully, I served as a Frontend Web Developer focusing on building highly performant, user-centric dashboards and interactive tools using modern React architectures.

# The Challenge
Prepfully needed to increase user engagement and time-on-site by providing a more interactive, seamless experience for candidates navigating their question banks and interview preparation materials. The existing UI was functional but lacked the polish and speed required to retain high-intent users.

# The Solution
I drove the frontend development of key user-facing features:
1. **User Dashboards**: I built responsive, intuitive user dashboards and secure admin tools using **React** and **Next.js**. The architecture leveraged server-side rendering (SSR) for blazing-fast initial page loads and SEO optimization.
2. **Interactive Question Bank**: I launched a dynamic question bank experience, featuring advanced filtering, instant search, and seamless pagination without full page reloads.

# Tech Stack & Skills
- **React.js & Next.js**
- **TypeScript & JavaScript (ES6+)**
- **HTML5 & Modern CSS3**
- **State Management & API Integration**

# The ROI & Impact
> **Increased user session time and search usage by 25%** immediately following the launch of the new interactive Question Bank experience.

> **Improved Core Web Vitals** and overall application performance by migrating key dashboard views to Next.js server-side rendering.
