# Overview
During my tenure at IEMA Research and Development, I operated as a Software Development Intern, taking on significant technical leadership responsibilities and deeply optimizing backend utilities.

# The Challenge
The company's core security utility, *IEMSecure*, was suffering from major performance bottlenecks. The data encryption and application scanning processes were running synchronously, causing unacceptable wait times for end-users when processing large datasets.

# The Solution
I took ownership of the IEMSecure performance refactor:
1. **Multithreaded Refactor**: I dove into the core Python utility and rewrote the synchronous scanning algorithms to utilize Python's multithreading and multiprocessing libraries.
2. **Team Leadership**: I led a cross-functional team of 5 interns, organizing sprint tasks and code reviews to successfully deliver multiple client web applications on schedule.
3. **CI/CD & Cloud Infrastructure**: I set up automated CI/CD pipelines to ensure code quality and utilized AWS (Amazon Web Services) for reliable, scalable deployment of the client web apps.

# Tech Stack & Skills

- **Web Application Development**
- **Agile Leadership & Code Review**
