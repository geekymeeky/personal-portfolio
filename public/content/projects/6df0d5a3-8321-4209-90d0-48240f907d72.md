# The Challenge
To demonstrate advanced frontend mobile engineering and state management capabilities, I set out to build a cross-platform weather application that rivaled the design and performance of native iOS and Android apps.

# The Solution
I developed **Mausam**, a stunning mobile application built entirely in **Flutter** and **Dart**.

The application fetches real-time meteorological data and forecasts using open REST APIs. I implemented a highly responsive, custom UI featuring a modern Glassmorphic and Neumorphic design system. The app handles asynchronous API calls, JSON parsing, and device geocoding natively.

Because of the extremely clean codebase, modular widget architecture, and optimized state management, the project was officially reviewed, approved, and featured on the **Google Dev Library**.

# Tech Stack
- **Flutter Framework**
- **Dart Programming Language**
- **REST APIs & JSON Parsing**
- **Geocoding & Location Services**

# The ROI & Impact
> **Featured globally on the Google Dev Library**, serving as an exemplary open-source reference architecture for other developers learning Flutter.

> Achieved a **zero-crash rate** by implementing robust try/catch error handling and graceful degradation when network connectivity drops.
