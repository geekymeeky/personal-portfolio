# The Challenge
Executive leadership needed a way to visualize loan origination revenue, pipeline health, and regional sales performance in real-time. Native Salesforce reports and standard dashboards were too rigid and couldn't provide the interactive, multi-layered data visualizations the business required to make rapid financial decisions.

# The Solution
I engineered a highly interactive **Revenue Dashboard** natively inside Salesforce. Instead of relying on expensive third-party BI tools like Tableau, I built a custom solution using **Lightning Web Components (LWC)** and integrated the open-source **Chart.js** library.

The dashboard pulls live data using optimized **SOQL** queries via Apex controllers. The UI allows executives to filter by date ranges, loan products, and sales regions instantly without page reloads.

# Tech Stack
- **Lightning Web Components (LWC)**
- **Apex Controllers (AuraEnabled, Cacheable)**
- **Chart.js (via Static Resources)**
- **SOQL & Aggregate Queries**
- **HTML5 & CSS3 (SLDS)**

# The ROI & Impact
> **Eliminated 5+ hours per week** of manual data export and Excel pivot-table generation previously done by the operations team.

> Provided real-time pipeline visibility, empowering the executive team to identify and close a 15% gap in Q3 revenue projections proactively.
